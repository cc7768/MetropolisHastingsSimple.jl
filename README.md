# Metropolis Hastings

## Intro

Provides simple code to do do Bayesian estimates using Metropolis-Hastings. Not meant to replace (or even compete with) `Lora.jl`; this is simpler code that is less general but I know how it works -- I eventually will figure out how `Lora.jl` works and once I do this repository will probably be deleted.
